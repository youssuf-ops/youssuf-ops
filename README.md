# 📊 Portfólio de Análise de Dados

## Análise de Capital Humano

Este relatório fornece uma visão geral da força de trabalho de uma organização, destacando indicadores como:

📌 Tamanho e composição da equipa  
📌 Distribuição geográfica  
📌 Diversidade de género  
📌 Estrutura organizacional  
📌 Turnover (rotatividade)

O objetivo é identificar oportunidades para melhorar a gestão de pessoas e apoiar decisões estratégicas.

## Análise de Vendas

Este estudo analisa o desempenho comercial de uma empresa, identificando tendências e oportunidades de crescimento.

Principais indicadores:

📌 Volume total de vendas: **1,23M**  
📌 Principal mercado: **EUA (895K)**  
📌 Segundo mercado: **RU (333K)**  
📌 Melhor trimestre: **Q1 (420K)**  
📌 Vendedor líder: **Pinto (226K)**

Recomendações estratégicas incluem:

- Analisar sazonalidade das vendas  
- Replicar boas práticas do vendedor líder  
- Avaliar expansão para novos mercados
